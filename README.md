# SCP - Containment Breach: No Drop Challenge
The SCP - Containment Breach: No Drop Challenge, is a challenge where the player is not allowed to drop any item they pick up during their playthrough.

This challenge can be played on two modes: Regular and Hard.
The challenge rules can be found in the file: Challenge Rules.
The scoring system to calculate your points can be found in the file: Scoring System.

Should the player break any rules or encounter an unsolvable situation during their playthrough, they have to start a new game on a blank map seed and reset their time.

Good Luck!
